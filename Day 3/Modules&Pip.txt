### **Modules in Python**
A **module** in Python is a file containing Python code (functions, classes, and variables) that can be reused in different programs. Modules help organize code and enhance reusability.

#### **Types of Modules**
1. **Built-in Modules**: Provided by Python (e.g., `math`, `random`, `os`).
2. **User-defined Modules**: Created by users to organize their code (`.py` files).
3. **Third-party Modules**: Developed by external contributors and installed via `pip` (e.g., `numpy`, `pandas`).

#### **Using Modules**
- **Import a Module**:  
  ```python
  import math
  print(math.sqrt(25))  # Output: 5.0
  ```
- **Import Specific Functions/Classes**:  
  ```python
  from math import sqrt, pi
  print(sqrt(16))  # Output: 4.0
  ```
- **Rename a Module (aliasing)**:  
  ```python
  import numpy as np
  ```
- **Import All Items (Not Recommended)**:  
  ```python
  from math import *  # Imports all functions
  ```

#### **Creating a Module**
Create a file `mymodule.py`:
```python
def greet(name):
    return f"Hello, {name}!"
```
Use it in another script:
```python
import mymodule
print(mymodule.greet("Alice"))
```

### **Pip in Python**
**Pip** (Package Installer for Python) is a tool used to install and manage third-party Python packages.

#### **Checking if Pip is Installed**
```sh
pip --version
```

#### **Installing a Package**
```sh
pip install numpy
```

#### **Upgrading a Package**
```sh
pip install --upgrade pandas
```

#### **Uninstalling a Package**
```sh
pip uninstall requests
```

#### **Listing Installed Packages**
```sh
pip list
```

#### **Checking Package Details**
```sh
pip show flask
```

#### **Installing from a Requirements File**
If you have a `requirements.txt` file:
```sh
pip install -r requirements.txt
```
